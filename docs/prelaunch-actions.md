## 1. X キーワード検索結果

### 検索ワードと発見

#### 「休日 マンネリ」「休日 同じ場所」

- 「休日マンネリ」系の記事が多数存在。ブログ・メディアでは定番テーマとして扱われている
- 「休日のデートがマンネリ」はカップル向けメディアの定番トピック → Roambleはカップルユースも想定可能

#### 「散歩 飽きた」「同じルート」

- **@kmizu** (X): 「散歩が飽きるってのはもちろんあって、そういうときは逆方向に周回したり、普段使ってないルート開拓したり、違う音楽聴いたりなどなど、退屈を紛らわす工夫をちょくちょくしてます。」
- **有吉弘行** (ラジオ): 「家の半径1kmが飽きた」 → まさにRoambleの解決する課題
- **NAVITIME**: 「周辺をぐるっと回って戻ってこれる散歩ルート機能」を開発 → 企業も課題を認識している
- ブログ記事: 「近所の散歩飽きた！マンネリ脱出へ5つの方法」「散歩に飽きた人におすすめの方法」等が複数存在

#### 「新しい場所 行きたい めんどくさい」「出かけたいけど行き先が決まらない」

- Yahoo!知恵袋: 「休みの日、家にいたくないけど、行きたい所も特にない時はどうしますか？」
- 複数のメディア記事:「出かけたいけど行き先が決まらない」「行きたいところがない」が定番の悩みとして扱われている
- 課題の構造: **「出かけたい気持ちはある」+「探すのが面倒/情報過多で疲れる」** → まさにRoambleのターゲット

#### 「新しい場所に行くのが怖い」「初めての店に入れない」

- Yahoo!知恵袋: 「[普段行かない店に入る勇気が出ないのですがどうしたら行けるようになりますか](https://detail.chiebukuro.yahoo.co.jp/qa/question_detail/q11253695226)」
- note記事: 「[初見の店にスムーズに入れない](https://note.com/merry_phlox5436/n/n7176018ee479)」— 事前にSNSやGoogle Mapsで何度も確認し、店の前を通過して中を観察し、それでも入れずに30分〜1時間迷う体験談
- 日経新聞: 「[カフェの店員がなぜ怖い…あなたが1人で入れない店](https://www.nikkei.com/article/DGXNASFE18058_Y4A310C1945M00/)」— 日経が記事にするほど一般的な悩み
- DANRO: 「[ひとりでバーに入れない 意外と険しい「大人の階段」](https://danro.bar/11852586/)」
- 課題の構造: **「行きたい気持ちはある」+「一人で初めての場所に入る心理的ハードルが高い」**。チェーン店は安心だが個人店は怖い。グループなら行けるが一人だと入れない
- **Roambleとの関係**: 「提案される」ことで自分で探す不安が軽減され、行動の背中を押す効果が期待できる。要件定義書の「適度な背中押し」コア価値に直結

### ユーザーの語彙（LPコピーに活用）

| ユーザーの言葉 | 活用方針 |
|---|---|
| 「飽きた」 | 最も直感的。LP見出しに使える |
| 「マンネリ」 | 共感度が高い。投稿向き |

| 「行きたい場所がない」 | 検索行動と一致。SEO的にも有効 |

| 「家の半径1kmが飽きた」 | キャッチーな表現。引用可能 |
| 「入る勇気が出ない」 | 心理的ハードルの言語化。共感投稿・LP課題説明に使える |
| 「初見の店に入れない」 | 一人行動の壁。具体的で共感度が高い |
| 「一人で入りづらい」 | チェーン店 vs 個人店の対比で使える |

### 判断

**課題は実在する。** ブログ記事やSNS投稿で繰り返し語られているテーマであり、NAVITIMEのような企業も関連機能を開発している。ただし「死ぬほど困る」課題ではなく「慢性的なモヤモヤ」であることも確認（要件定義書の分析通り）。

さらに「新しい場所に行くのが怖い」「初めての店に一人で入れない」という心理的ハードルも広く共有されている課題であり、日経新聞が記事にするほど一般的。Roambleの「提案される → 行動の背中を押す」という価値提案は、この心理的障壁の軽減にも効く可能性がある。

## 2. X 共感投稿（草案）

### 投稿方針

- ユーザーの普段の文体に合わせる（カジュアル・タメ口・自然体）
- 「使いますか？」系は避け、課題の共感を問う
- 個人の体験・実感ベースで書く
- **メイン軸は「新しい場所に行くのが怖い」**。「めんどくさい」は副次的な要素として扱う

### 投稿案1（課題の言語化テスト — 投稿済み）

> **投稿済み**: 2026-02-07
> **URL**: https://x.com/Hiru_ge/status/2020004905256382759

```
気になってる店の前をソワソワしながら素通りしちゃう問題解消したい
一人で知らない店に入るハードルが地味に高くて、チェーン店に行ってしまいがち
```

### 投稿案2（Build in Public 導入）

```
「新しい場所に行きたいけど怖い」を解決するWebアプリ作り始めた

自分がまさにこれで、知らない店の前を素通りしちゃうタイプ。「ここ行ってみたら？」って背中押してくれる存在がほしくて、ドッグフーディング前提で作ってる

コンフォートゾーンから一歩出る体験をゲーム的に積み重ねられるやつにしたい
```

### 投稿案3（内省型 — コンフォートゾーンの言語化）

```
散歩してて「あ、この路地気になる」って思っても曲がれないことがある。別に危険でもなんでもないのに、知らない道に入るのがなんか怖い

これがコンフォートゾーンってやつなんだろうな。生活圏って思ってるより狭い
```

### 投稿の使い方

1. **投稿案1を最初に投稿** → 「知らない店に入れない」共感の反応を見る
2. 反応があれば **投稿案2** で自然にプロダクト紹介 → LP誘導の布石
3. **投稿案3** は反応次第で投稿（散歩 × コンフォートゾーンの気づき系）
4. 反応が薄ければ「怖い」の言語化を変えて再トライ（「勇気がいる」「ハードルが高い」等）

### 観察ポイント

- いいね/RT数 → 共感度の定量指標
- リプの言葉遣い → 「怖い」をユーザーがどう表現するか（LP コピーに直接活用）
- 引用RTの文脈 → 「自分も〜」系の追体験が出ればニーズ確認
- **特に注目**: 「わかる」系のリプが来るか。来れば課題フレーミングとして正しい

#### セクション1: 課題の共感

**見出し**: こんな経験、ありませんか？

- 気になる店があるのに、一人で入る勇気が出ない
- 知らない道に曲がるのが、なんとなく怖い
- 結局いつものチェーン店、いつもの散歩ルートに落ち着く
- 「新しい場所に行きたい」のに、いざとなると足が動かない
- 行動範囲が思ったより狭いことに気づいて、ちょっとモヤモヤする

---

#### セクション2: 解決策

**見出し**: Roambleがやること

1. **背中を押す** — 現在地の近くから「まだ行ったことのない場所」を提案。自分で探す不安がなくなる
2. **一歩を記録する** — 行ってみたら「行った！」ボタン。小さな冒険が記録に残る
3. **広がりを実感する** — 訪問履歴で、自分の行動範囲が広がっていくのが見える

※ 将来的にはXP・レベル・バッジなどのゲーミフィケーション要素を追加し、コンフォートゾーンを出る体験をゲーム的に積み重ねられるようにする予定
