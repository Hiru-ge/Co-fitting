## 3. 肉声スクリプト（話す内容）

> スマホのボイスメモで一発録りでOK。「喋ってます」感はむしろ正義。

```
（0〜3秒）
自作アプリを、作った本人が毎日開けていなかった。

（3〜8秒）
近くの知らない場所を毎日3件提案して、
行くたびに経験値がもらえるアプリなんですけど、

（8〜15秒）
提案は毎朝来てるのに、
通知がないから開くのを忘れてて。

（15〜22秒）
だから通知機能をつけることにしました。
作った人間がこれです。

（22〜27秒）
（無言 / CTAテロップのみ）
```

**録音のコツ**:
- 普通のトーンで喋る。「話してます」感が出るのが正解
- テイクは2〜3本撮って一番自然なものを選ぶ
- BGMと被らないよう静かな場所で録る
